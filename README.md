**Автор:** Лубенец Захар

**Дата сдачи:** 07.05.2026

## Описание программы

Book Tracker — это кроссплатформенное GUI-приложение для учёта прочитанных книг. Позволяет быстро добавлять книги, просматривать их в табличном виде, применять фильтрацию по жанру и объёму, а также сохранять и загружать данные в формате JSON.

## Требования для запуска

- Python 3.10 или выше
- Tkinter (обычно идёт в комплекте с Python)

## Как запустить

```bash
git clone https://github.com/zaharlubinec11-afk/Book-Tracker
cd Book Tracker
python main.py
